# 俄罗斯方块 - 移动端糖果风

## 📋 项目需求
- **目标平台**：移动端（手机浏览器）
- **视觉风格**：糖果色系（粉色、蓝色、黄色、绿色等柔和色调）
- **核心功能**：基础俄罗斯方块 + Ghost预览 + 消除动画

## 🎯 技术实现要点

### 核心数据结构
```javascript
board[20][10]     // 游戏棋盘（20行10列）
currentPiece      // 当前方块 {type, rotation, x, y}
nextPiece         // 下一个方块
score             // 分数
level             // 等级
lines             // 消除行数
```

### 移动端优化
- `touch-action: none` 禁止默认触摸行为
- `preventDefault()` 防止页面滚动
- 虚拟按键使用 `touchstart` + `touchend` 事件
- Canvas自适应屏幕宽度

### 性能优化
- 使用 `requestAnimationFrame` 游戏循环
- 离屏Canvas预渲染方块
- 减少不必要的重绘

## 🎮 使用说明

### 电脑端
- **← →** 左右移动
- **↑** 旋转方块
- **↓** 加速下落
- **P** 暂停/继续

### 移动端
- 点击虚拟按钮控制
- 长按可连续执行
- 支持暂停/继续功能

### 游戏规则
- 填满一行自动消除并得分
- 消除1行得100分，2行300分，3行500分，4行800分
- 每10行升一级，速度加快
- 方块堆到顶部游戏结束
